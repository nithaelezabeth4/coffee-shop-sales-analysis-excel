# Coffee Shop Sales Analysis Excel

This project presents an interactive **Coffee Shop Sales Analysis Dashboard** created in **Microsoft Excel**.  
The dashboard helps analyze sales performance, customer ordering patterns, product category distribution, and store location performance.

## Project Overview

The dashboard provides insights into:

- **Total Orders:** 149,116  
- **Total Quantity Sold:** 214,470  
- **Total Sales:** $698,812  
- **Average Bill:** $4.69  

It also includes visual analysis of:

- Sales distribution by product category
- Top 5 products by bill amount
- Quantity ordered based on hours
- Daily order trends
- Sales distribution by product size
- Orders and quantity distribution by location

## Dashboard Features

- Interactive **slicers** for:
  - Day Name
  - Month Name
  - Hour

- KPI Cards for:
  - Total Orders
  - Total Quantity
  - Total Sales
  - Average Bill

- Charts used:
  - Pie Charts
  - Bar Charts
  - Line Chart
  - Clustered Column Chart

## Tools Used

- **Microsoft Excel**
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Formatting

## Key Insights

- Coffee category contributes the highest share in product sales.
- Peak quantity orders happen during morning hours.
- Sales show a rising trend from January to June.
- Lower Manhattan shows strong order and quantity performance compared to other locations.
- Top products contribute significantly to overall revenue.
